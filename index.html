Essential formulas, definitions, and algorithms from Chapter 6

Core Formulas
Support (relative) s(A → B) = P(A ∪ B) = support_count(A ∪ B) / N
Confidence c(A → B) = P(B|A) = support_count(A ∪ B) / support_count(A)
Lift (Correlation Measure) lift(A, B) = P(A ∪ B) / [P(A) · P(B)] = conf(A → B) / sup(B)
< 1: negatively correlated · = 1: independent · > 1: positively correlated
